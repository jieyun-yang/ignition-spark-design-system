# Rich Text Editor

## Overview

The Rich Text Editor is a component for authoring and formatting text content. It provides tools for creating rich, structured text including placeholders, HTML source editing, tables, and text highlighting.

## When to Use

- Creating terms, email, and message templates (including placeholders)
- Writing messages and emails in proposal editor (when not using templates)
- Editing service descriptions in the service library

## When NOT to Use

- Simple text input fields that don't require formatting

## Features

| Feature | Description | Notes |
|:--------|:------------|:------|
| **Text Selection** | Select text to apply formatting | - |
| **Source View** | View underlying HTML code | ONLY for internal Ignition users |
| **Undo/Redo** | Revert or reapply recent changes | - |
| **Placeholder Insert** | Insert predefined placeholders | - |

## Best Practices

### Do's
- Use in areas where text content can or needs formatting (bold, italics, lists, headers)
- Provide clear placeholder insertion for templates

### Don'ts
- Don't use for basic input fields where formatting is unnecessary
- Don't add complexity where simple text input suffices

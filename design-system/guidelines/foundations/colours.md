# Colours

## Overview

This document outlines the foundational colour palette used across the design system. Adhering to these defined colours ensures visual consistency, reinforces brand identity, and contributes to a cohesive user experience.

## Colour Categories

### Text Colours
Designated colours for all textual content, optimized for readability and information hierarchy.

### Border Colours
Colours specifically allocated for borders, outlines, and separators within UI components.

### Primary Colours
Core brand colours used for primary actions, branding elements, and key informational highlights.

- **Dark Green**
- **Purple** (primary action colour)
- **Orange** (currently used for primary CTAs)

### Gray Colours
Versatile neutral gray tones for:
- Backgrounds
- Secondary text
- Disabled states
- General UI elements

### Secondary Colours
Extended palette complementing primary colours:
- **Yellow**
- **Green**
- **Red**
- **Blue**
- **Teal**
- **Pink**

### Marketing Gray
A distinct gray variant specifically for marketing and promotional materials.

## Usage Guidelines

### Apply colours by purpose:
- **Text colours** - For all textual content
- **Border colours** - For outlines, dividers, container borders
- **Primary colours** - For primary actions, branding, key highlights
- **Secondary colours** - For supplementary actions, status indicators, decorative elements
- **Gray colours** - For neutral backgrounds, subtle text, disabled states

## Best Practices

### Do's
- Always use predefined colour tokens instead of arbitrary HEX/RGB values
- Apply colours according to their intended categories
- Prioritize primary colours for key branding elements and actions
- Ensure sufficient colour contrast

### Don'ts
- Don't create custom colours outside the palette
- Don't use colours inconsistently (e.g., red for success)
- Don't rely solely on colour to convey meaning

## Accessibility

- Ensure minimum contrast ratio of **4.5:1** for normal text
- Ensure minimum contrast ratio of **3:1** for large text (WCAG 2.1 AA)
- Always supplement colour with text labels, icons, or patterns
- Test colour palettes with colour blindness simulators
